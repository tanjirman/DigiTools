## ✨ Features

### 🛒 Smart Shopping Cart
Add, remove, and manage your selected digital tools in a sleek cart system — with live item count displayed in the navbar and instant toast notifications on every action including add, remove, and checkout.

### 📦 Dynamic Product Catalog
Browse 8+ premium digital tools displayed in a clean 3-column grid layout, each card includes pricing, period, feature list, color-coded tags (Popular, New, Best Seller), emoji icons, and a one-click Buy Now button that disables once added.

### 💳 Transparent Pricing Plans
Choose from Starter, Pro, and Enterprise plans with clear feature breakdowns and a highlighted Most Popular badge — designed to fit individuals, professionals, and growing teams alike.

### 🔁 Tab-Based Navigation
Switch seamlessly between the Products and Cart sections using smooth tab buttons. The active tab is fully controlled by React state — clicking the cart icon in the navbar automatically jumps to the cart tab.

### 📌 Sticky Navbar
The navigation bar stays fixed at the top while scrolling — always giving users quick access to the cart count, login, and get started button no matter how far down the page they are.

### 🎉 Toast Notifications
Every key user action triggers a real-time toast notification using React Toastify — adding to cart, removing an item, and completing checkout all give instant visual feedback.

### 📱 Fully Responsive Design
The entire layout is mobile-first and adapts beautifully across all screen sizes — from single-column cards on mobile to a full 3-column grid on desktop.

### 🧹 Checkout & Reset
Clicking Proceed to Checkout clears the entire cart, switches back to the Products tab, and shows a success notification — giving users a clean and satisfying end to their shopping flow.
